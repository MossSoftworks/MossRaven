## 5. Environment gotchas (learned today — don't relearn them)

- **One git writer at a time.** The Cowork session and Claude Code must not run git mutations in this worktree concurrently.
- **PowerShell 5.1** (this box): strips a bare `--` token — pass test args via `cmd /c "...""` ; and mangles inline JSON to native exes — backslash-escape inner quotes (`'{\"k\":\"v\"}'`). Both already fixed in `scripts\`.
- **Native tools write progress to stderr** (cargo, dotnet, our service) — don't let `$ErrorActionPreference='Stop'` treat that as failure (fixed in `Invoke-Native`).
- **`dist\` lock:** a running MCP daemon / WPF shell holds `mossraven-service.exe`; see [3].
- **No keys committed.** `CEREBRAS_API_KEY` is set in Taylor's shell (Tier-2 live). `MOSSRAVEN_ANTHROPIC_API_KEY` is NOT set → Tier-1 is Mode B by default. Never put `ANTHROPIC_API_KEY` in the Claude Code shell (flips billing).
- **Never commit `vendor/`** or GGG data. `/data/`, `scripts/*.log`, fixture `_*.json` are gitignored.
